# Splatoon3-Toolbox
A fork of Switch Toolbox to revolve around Splatoon 3 model editing

### Download here:
https://github.com/cakezara/Splatoon3-Toolbox/releases

This tool adds many features + tools for quality of life and speed, specifically in importing custom/ported models.
Credits for tools supplied are below.

## Added Features

- Batch Replace Materials (with preset mats too!)
- Auto Texture Replacement
- Alb to Opa
- Spm to Rgh

More features are to come as I am planning to update this semi-often!
